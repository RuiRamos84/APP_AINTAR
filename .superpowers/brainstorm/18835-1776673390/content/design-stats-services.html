◎ Progress Rings animados STATS
SVG stroke-dashoffset animado via Framer Motion ao entrar no viewport. Cada ring tem delay escalonado (0.1s, 0.25s, 0.4s, 0.55s). Glow pulse contínuo em fundo.
💧 Liquid ripple hover SERVIÇOS
CSS ::before com radial-gradient que expande de 0 para 300px no hover. Cria efeito de "gota de água" a espalhar-se. Zero JS, pure CSS.
✂ Clip-path reveal no scroll
Cards entram com clipPath: inset(0 100% 0 0) → inset(0 0% 0 0) via Framer Motion whileInView. Efeito de "cortina lateral" a revelar.
⚡ Accent neon do B
No hover dos cards, o icon-wrap ganha box-shadow: 0 0 16px rgba(32,180,220,0.35). Glow subtil que reforça a paleta C+B sem exagero.
